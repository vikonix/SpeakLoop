# SpeakLoop

AI-powered language learning through real conversation practice.

## About

SpeakLoop is an MVP for practicing foreign languages with AI.  
The app focuses on speaking, vocabulary, and natural conversations instead of traditional exercises.

## Features

- AI conversation partner
- Real-time language practice
- Vocabulary improvement
- Grammar correction
- Personalized learning experience

## Tech Stack

- Python 3.11.9
- FastAPI
- OpenAI API

## Getting Started

```bash
git clone https://github.com/yourusername/speakloop.git

cd speakloop

python3.11 -m venv .venv
source .venv/bin/activate

pip install -r requirements.txt
python main.py
